Welcome to the Fair Allocation App

This application allows you to explore algorithms for the fair allocation of indivisible goods.

About Fair Allocation

The problem of fair allocation of indivisible goods involves distributing a set of items to a set of agents in a way that is considered fair according to certain criteria.

Existing Algorithms

Several algorithms have been developed to address this problem, each with its own approach to achieving fairness.

To learn more about these algorithms, select one from the "Algorithm" menu above.

Ready to try it out? Click on "Run Algorithm" to get started.
